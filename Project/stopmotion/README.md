# 춤추는 옷 만들기 (2인 프로젝트)

옷이 공중에 떠다니는 영상을 딥러닝을 활용하여 만드는 것이 저희 팀의 목표입니다.<br><br>
역할: unet모델구현, 데이터제작 및 데이터라벨링<br><br>
딥러닝 모델은 U-NET을 사용하였습니다.<br><br>
(tensorflow, keras, python 사용)
